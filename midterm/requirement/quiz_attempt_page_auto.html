Description and tasks:

Students have been asked to create an android health app to show the “Calorie burned” while doing various exercises for one hour.

 

The Main activity (Exercise activity) allows the user to select different types of exercise from a ListView or a Spinner as shown in the image (screenshot) below. If your first name starts with a letter from A-M use a list view, otherwise use a spinner – (N-Z).

 

Based on your exercise selection (for example: Jumping Rope, Swimming, Bike Riding, Walking and Running), When you double click on the selected item, which will display a graph in the second activity. (Screenshots shows a graph for the swimming activity)

 

The second activity (graph activity) should draw the graph for exercise selection using an ImageView as a drawing canvas. The ImageView height and width should be 500 and 900 respectively (You can customize the canvas size based on the screen size) and the background color should be black. Use drawRect() method to draw these graphs.

 

Use different colors to draw graph as given below:

 

  • Jumping Rope - Blue
  • Swimming - Red
  • Bike Riding - Yellow
  • Walking – Green
  • Running - Purple

 

Every rectangle, which represents the exercise, should be labelled in white color (text color).

 

You should create appropriate resources files to enhance your app. You should store the exercise values such as Jumping Rope, Swimming, Bike Riding, Walking and Running in the string file.

 

As shown in the graph activity, you should have legend to show how much kcal burned for each exercise.

 

  • Jumping Rope – 750 kcal
  • Swimming – 600 kcal
  • Bike Riding – 500 kcal
  • Walking –  300 kcal
  • Running – 600 kcal

                          

                                     

Students can use virtual or connected devices to run the solutions. The results or output must be captured (screenshots) and attached to the submission folder.
